Work completed:

  • learned how Wordpress works
  • make banner images stretch across the inner div
  • researched the slideshow plugin
  • fixed color/opacity on several slideshows
  • created popup for the volunteer form
  • created links within the FAQ pages
  • created 3 contact forms using a plugin, each with some javascript/css modifications
  • taught how to use HTML tables
  • adjusted many CSS thingamajigs
  • made text flow around quote blocks, & made the quote blocks easily resizable
  • created responsive image map for the "communities" homepage

Challenges faced:

  • This is my first real project. As such, I’m still just learning how web design works. It’s exciting! But I hope I’m doing a decent job.
  • I’m working remotely: I’m in the U.S. while everyone I’m working with is in Peru. Fortunately, we are in the same time zone. On the upside, my written communication skills are constantly improving.
  • I haven’t been able to figure out how to upload JavaScript files to their server, so all scripts are written inside the HTML.
  • It is uncertain whether the CTTC will have the help of another programmer when they want to revise this site. Thus, I need to make sure everything is as user-friendly as possible.
